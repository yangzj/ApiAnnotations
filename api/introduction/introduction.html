    ApiAnnotations框架是一套拘于java语言的web应用开发框架，底层应用原生态的servlet 3.0、spring 4.0、hibernate，可以说是站在巨人肩膀上的创新，它具有富于表达性且简洁的语法。 我们致力于实现开发过程的快速、代码的简洁、功能的复用性，用最少的代码解决繁琐的问题。我们相信，使用我们的框架，开发过程应是愉快、创造性的体验。

ApiAnnotations框架的目标是给开发者创造一个愉快的开发过程，处理平常开发繁琐枯燥的问题，并且不牺牲功能的稳定性。让开发者把刀用在刀刃上，用更多的精力写出更棒的代码。 为了实现这个目的，我们回归到原生态的servlet，以及吸收开源框架spring、hibernate的精华，博取众框架之长处运用于ApiAnnotations框架之中。

ApiAnnotations框架是容易理解并且简洁强大的，能够高效的开发出安全、健壮、稳定的应用。 通过平常枯燥的注释功能运用于开发中，并且提供一套简洁的表达式，解决跨域问题，让代码更好维护，并且提供一套强大的API控制台，让单元测试，项目任务分配更加的简单易用。

学习

学习ApiAnnotations框架，首先需要读懂这个整个文档，并且java基础好的开发者，可以直接学习我们的开源的源码。 直接读懂源码，更能学会整个框架的底层。有疑问可以直接联系我们。

特征

  • 强大的后端校验功能，简化数据检验步骤
  • 规范化的javadoc风格
  • 安全和规范化的实体模型
  • 根据注解和javadoc，编译时自动生成代码
  • 支持跨域访问
  • 功能齐全，拥有spring，hibernate，mybatis全部功能
