  • 学习ApiAnnotations框架

介绍

自动生成代码工具是ApiAnnotations框架的最大一个特色，节省了重复、枯燥、繁琐这部分代码的编写。用更多的精力写出更棒的代码。

当开发者在eclipse配置好 > Annotation processing后，项目将在根目录自动生成apt_generated文件夹， 自动生成的内容从cn/banbang/{项目名称}里面的代码开始自动生成。所有的项目功能实现都会生成在此文件夹里面，这样 能够保证了实现类的干净和简洁，不需要人工维护，一旦项目实现类有修改，都会自动生成。

注：apt_generated里面自动生成的代码不能修改
