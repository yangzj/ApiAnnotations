创建第一个ApiGroup

  • 简介
  • 添加FirstAction接口组
  • 创建FirstAction接口组实现类FirstActionImpl

简介

到这里，说明你的开发环境已经配置好，并且你的项目已经使用上了ApiAnnotations框架。接下来，我们赶紧来实现一个实例！ 如果还没有配置环境，请先阅读配置说明。

  • 创建一个Action接口组：FirstAction，返回值为“hello word！”
  • 在Action类中使用@ApiGroup自动生成Servlet类以及数据校验类
  • 并且创建FirstAction实现类：FirstActionImpl，具体实现查看以下代码。

添加FirstAction接口组

在项目cn/banbang/demo/action包下创建接口文件FirstAction.java文件，并且设计了接口：helloWorld，内容如下： 


import cn.banbang.ApiAnnotations.core.annotation.ApiGroup;

/**
 * @author banbang
 * @description 创建第一个ApiGroup
 * @id 101
 */
@ApiGroup
public interface FirstAction {
	/**
	 * 第一个ApiGroup
	 * @id A10101
	 */
	public String helloWorld();
}

那么http://localhost:8082/demo/A10101就是为该接口，http请求的入口。

注：1、这是一个interface类；
2、类的javadoc和method的javadoc为必填项，不可忽略他，不然将编译无法通过。而且这些javadoc是有严格要求的，等下会提到这个；

保存Action类（Eclipse将自动编译，且在apt_generated自动生成servlet经典实现类）

创建FirstAction接口组实现类FirstActionImpl

在项目cn/banbang/demo/action/impl包下创建FirstAction接口类的实现类FirstActionImpl，代码内容如下 

import javax.transaction.Transactional;

import org.springframework.stereotype.Service;

import cn.banbang.demo.action.FirstAction;

@Transactional
@Service
public class FirstActionImpl implements FirstAction {

	@Override
	public String helloWorld() {
		return "hello world!";
	}
}

注：实现类注解使用的注解@Transactional与@Service，是spring提供的。
