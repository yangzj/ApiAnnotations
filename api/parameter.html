注释说明

interface头部注释说明

@author 作者，例如：小杨
@description 类描述，例：用户角色类
@id 每个类都有唯一一个自定义id标识，例：106为UserAction的id

interface注解

@Query 自动生成查询接口实现类
@ApiGroup 自动生成Servlet类以及AOP数据校验类

interface方法注释描述

@id 接口标识，必填且不能重复，格式为：A+接口组id+两位数字
@descriptions 接口描述/依次接口参数描述
@permissions 访问权限
@return 接口返回值
@re.telephone 参数

interface方法注释描述

@name 对前端传入参数进行改名
@scope database、request、session、cookie、header、ServletContext
@default 参数设定默认值
@regex 参数正则表达式验证

queryType参数详解(sql为sql语法，hql为hql语法)

@sqlGet / @hqlGet 返回一个数据对象
@sqlList / @hqlList 返回一个数据列表
@sqlPage / @sqlPage 返回一个数据列表带分页，数据结构：data{list, total}
