下载

在平台上下载ApiAnnotations 0.1框架jar源码，去下载。

maven配置

主要配置apiannotations-core-0.1-bin.jar和apiannotations-web-0.1-bin.jar两个jar包： 
<dependency>
	<groupId>cn.banbang.apiannotations</groupId>
	<artifactId>apiannotations-core</artifactId>
	<version>0.1</version>
</dependency>
<dependency>
	<groupId>cn.banbang.apiannotations</groupId>
	<artifactId>ApiAnnotations-web</artifactId>
	<version>0.1</version>
</dependency>
