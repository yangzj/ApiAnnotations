创建项目/导入项目

在配置Eclipse之前，需确保满足以下两点：

  • 在项目annotation配置之前，需根据说明先创建新maven项目/导入已有maven项目。
  • 执行java -version查看jdk，确保你的jdk版本至少在version 1.7.X或者以上

Eclipse配置

请注意，检查是否已经下载了一个重要的jar包：

  • 是一个注解处理器， 它只需要Eclipse编译的时间，但它需要配置在编译的classpath。

配置annotation processing

  • 在工程中根目录下创建compile-libs，将 复制到compile-libs该包里面，注意：不要将 复制到libs。
  • 右击你的项目，选择 》 Properties
  • 前往Java Compiler，钩选Enable project specific settings， 并检查并确保Compiler compliance level为1.7，否则注解处理器将不能运行；
  • 前往Java Compiler 》 Annotation Processing钩选Enable project specific settings
  • 前往Java Compiler 》 Annotation Processing 》 Factory Path，钩选Enable project specific settings，
  • 点击Add JARs添加，添加后之后，勾选为Annotation Processor
  • 设置完之后，保存修改。观察Eclipse的workspace是否有重新加载，如果有，说明配置成功。
  • 到此之后，你可以开始使用ApiAnnotations框架啦
注意：
1、不要将 复制到libs目录下，而是自己创建的compile-libs包下；
2、检查并确保Compiler compliance level为1.7，否则注解处理器将不能运行；
