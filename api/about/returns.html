过滤JSON对象属性输出

  • 简介
  • 过滤表达式
  • 模拟场景

简介

过滤表达式主要是为了控制数据的输出，为了保证某些数据不让用户看到，在后台做了过滤，这样能够保证数据的安全性。

过滤表达式

表达式符号

元字符 描述
* 表示输出同一级的所有值
; 表示不同级的值
| 表示属于同一级的属性

模拟场景一

♦ 当要输出所有json值时：

表达式：@returns ** 或者 @returns *;list.*;list.area.*

输出结果：

	"total": 100,
	"list": [
		{
			"realname": "小玲",

			"area" : {
				"province": "福建",

模拟场景二

♦ 当要输出部分json值时：

表达式：@returns *;list.id|realname|nickname|loginTime;list.area.city

输出结果：

	"total": 100,
	"list": [

			"id": "100000001"
