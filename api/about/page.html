分页使用

在很多框架中，分页有时很痛苦，但是ApiAnnotations框架的分页很简单，只要传入页数和页码就可以分页的查询数据。 提供两种查询数据的sql语法和hql语法，然后结合velocity的语法， 就可以实现简单的数据查询分页了。

以下为一个简单的分页的简单例子，代码如下：

/**
 * @author yzhujue
 * @description 用户查询类
 * @id 101
 */
@Query
@ApiGroup
public interface UserQuery {

	/**
	 * 分页用户信息列表
	 * @id Q10101
	 * @descriptions 页码/每页数量
	 * @filters open
	 * @pattern *;list.id|registerTime|nickname|realname|telephone|
	 * @queryType sqlPage
	 * @query
	 * 		<pre>
	  select * from TB_USER
	  where lifeCycle='AVAILABLE'
	  order by  "registerTime" desc
	 *       </pre>
	 */
	public Page<User> queryUserInfo(int pageSize, int pageNumber);
}

分页查询其实与普通的查询大同小异，只是javadoc标识中的@queryType属性为@sqlPage或者@hqlPage。 数据输出值，格式为：{"total": 0, "list": []}，具体的数据输出值表达式可查看数据返回表达式。
